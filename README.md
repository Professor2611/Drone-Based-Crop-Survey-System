# Drone Based Crop Survey System

## Overview

This project focuses on developing a drone-assisted agricultural monitoring system for crop surveying and precision agriculture applications.

## Problem Statement

Traditional crop inspection methods are time-consuming and inefficient. Drone-based surveying enables rapid field monitoring and efficient data collection.

## Objectives

- Perform aerial crop surveys
- Capture field images
- Monitor crop conditions
- Support precision agriculture
- Reduce manual inspection

## Components Used

### Flight System
- Pixhawk Flight Controller
- GPS Module
- Telemetry Module
- Brushless Motors
- ESCs
- LiPo Battery
- Propellers

### Survey System
- Camera Module
- Raspberry Pi
- Ground Control Station

## Methodology

1. Plan mission using Mission Planner.
2. Upload mission to Pixhawk.
3. Execute autonomous flight.
4. Capture field images.
5. Analyze collected data.

## Applications

- Precision Agriculture
- Crop Health Monitoring
- Yield Estimation
- Irrigation Planning

## Future Scope

- AI-based disease detection
- Computer vision integration
- Automated crop classification
- Real-time analytics
